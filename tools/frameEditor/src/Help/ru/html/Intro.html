Добро пожаловать в систему PDDResolver!

Эта система занимается распознаванием различных ситуаций на перекрёстке, для каждого участника дорожного движения выводит дальнейшие его действия, руководствуясь при этом базой знаний, описанной языком фреймов

Как ситуация на перекрёстке, так и редактирование базы знаний происходит в дружественном пользователю интерфейсе. К сожалению, пока имеется некоторый набор ограничений в обоих режимах работы в системе. Эти ограничения перечислены в главах, посвящённых этим режимам.

Глобальные сочетания клавиш

  • Ctrl-N - создать новый проект
  • Ctrl-S - сохранить проект
  • Ctrl-Shift-S - сохранить проект под другим именем
  • Ctrl-O - открыть проект
  • Ctrl-W - закрыть проект
  • Ctrl-Q - выйти из системы
  • Ctrl-1 - обычный режим работы
  • Ctrl-2 - режим редактирования базы знаний
  • F1 - вызов справки
