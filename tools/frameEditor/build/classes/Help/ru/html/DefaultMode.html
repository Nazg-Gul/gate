Обычный режим работы - такой режим, при котором пользователь задаёт некоторую ситуацию на перекрёстке и затем имеет возможность просмотреть в реальном времени смоделированное движение участников

Работа с визуальным редактором
Общий вид визуального редактора ситуации на перекрёстке отображён на следующем рисунке:

Он состоит из двух частей:
  • Изображение перекрёстка

    Отображает ситуацию на перекрёстке. Транспортные средства, знаки, светофоры помещаются здесь посредствам использования технологии drag-and-drop из палитры компонентов.

    После нажатия на кнопку "Моделировать" транспортные средства начнут своё движение по этому перекрёстку.

  • Палитра компонентов

    Хранит набор компонент (транспортные срадства, светофоры, знаки), которые можно помещать на перекрёстке посредствам технологии drag-n-drop.

Текущие ограничения, связанные с неполнотой реализации
К сожалению, работа над этим режимом ещё не завершена, поэтому существует некоторый ряд ограничений по описанию ситуаций на перекрёстке:

  • Отсутствуют знаки и светофоры
  • Нет возможности перемещать объекты, уже добавленные на форму
